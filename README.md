# RRDCS — Robot-Resistant Development Control System

**RRDCS** — это комплексная система автоматизированного контроля качества разработки программного обеспечения, разработанная для защиты кодовой базы от рисков, вызванных использованием ИИ-ассистентов (GitHub Copilot, ChatGPT, Claude и других LLM).
